日常工具
======

### 简单介绍
  * android一些日常工具的封装、okhttp网络框架的二次封装、安全加密工具封装、布局控件等等
  * 目的是让开发者简单的快速的集成，减少不必要的工作量

### 依赖版本 
  ![](https://jitpack.io/v/natural1law/briefness.svg "依赖版本")
### 使用依赖
  * **implementation 'com.github.natural1law.briefness:tag'**
  tag换成版本号即可
